# Football Accumulator Bot

A Python bot that scrapes football odds, builds 3–4 odds accumulators, logs results, and emails daily reports.

## 🚀 Features
- Scrapes upcoming matches and odds.
- Builds accumulators with target total odds between 3.0–4.0.
- Logs daily accumulators.
- Weekly performance summary.
- Sends 3 accumulator emails per day.
